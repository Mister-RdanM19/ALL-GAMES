Selamat datang di **ALL-IN-ONE GAMES**, sebuah mega-project yang mengumpulkan **21 Game Interaktif** dalam satu dashboard dashboard modern. Seluruh koleksi ini dibangun dengan fokus pada performa tinggi, desain *Neon-Cyberpunk*, dan pengalaman pengguna yang mulus tanpa perlu instalasi tambahan.

---

## üåü Fitur Utama
- **21 Game Eksklusif**: Mulai dari arcade klasik hingga simulasi bisnis kompleks.
- **Modern UI/UX**: Tampilan *Dark Mode* dengan aksen Neon yang konsisten.
- **Physics Engine**: Integrasi `Matter.js` untuk simulasi fisika realistis pada game tertentu.
- **Hacker Terminal Style**: Interface khusus pada game Programming Quiz.
- **Fully Responsive**: Nyaman dimainkan di berbagai perangkat browser.
